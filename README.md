# LUG-Wine-Experimental
Wine runners based on [TKG builds](https://github.com/Frogging-Family/wine-tkg-git) with LUG Community [patches](https://github.com/starcitizen-lug/patches) and additional experimental patches specifically for Star Citizen.

# Installation
The easiest way to install these runners is with the [LUG Helper](https://github.com/starcitizen-lug/lug-helper).

> [!NOTE]
> Minimum `glibc` requirement for these runners: `2.39` released 2024-01-31

# Versions
**lug-wine-tkg-experimental-git**  
Standard Wine built for maximum compatibility.

**lug-wine-tkg-staging-experimental-git**  
Wine plus staging patches.

**lug-wine-tkg-staging-experimental-wayland-git**  
Wine plus staging patches. Forces Wine Wayland by default.
