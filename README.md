# LUG-Wine
Wine runners based on [TKG builds](https://github.com/Frogging-Family/wine-tkg-git) with LUG Community [patches](https://github.com/starcitizen-lug/patches) and experimental patches specifically for Star Citizen.

# Installation
The easiest way to install these runners is with the [LUG Helper](https://github.com/starcitizen-lug/lug-helper).

> [!NOTE]
> Minimum `glibc` requirement for these runners: `2.39` released 2024-01-31

# Versions
**lug-wine-tkg-git**  
Standard Wine built for maximum compatibility.

**lug-wine-tkg-staging-git**  
Wine plus experimental staging patches.
